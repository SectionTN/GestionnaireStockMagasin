Configuration et lancement

Pour faire tourner l'application, il faut un projet Appwrite Cloud, Maven et Java 17.

1. Prerequis logiciels

  • JDK 17 ou plus recent
  • Maven 3.9 ou plus recent
  • Un compte sur cloud.appwrite.io

2. Creer le projet Appwrite

  1. Se connecter a Appwrite Cloud puis cliquer sur Create project.
  2. Donner un nom (par exemple "Magasin"). Noter le Project ID affiche sur la page d'accueil du projet.
  3. Dans Settings → API keys, creer une cle. Cocher les scopes : tables.read, tables.write, rows.read, rows.write. Copier la cle immediatement (Appwrite ne la montre qu'une fois).

3. Creer la base et les tables

  1. Aller dans Databases puis creer une base nommee magasin_db. Noter le Database ID.
  2. Dans cette base, creer quatre tables avec les colonnes ci-dessous. Verifier que les noms de colonnes correspondent exactement a ceux du code (sensible a la casse).

nomString255oui

prixDouble—oui
fournisseurString255non

Table commande

ColonneTypeTailleRequis
produit_idString36oui
quantiteInteger—oui
date_commandeString32oui

Table utilisateur

ColonneTypeTailleRequis
usernameString64oui
passwordString255oui
roleEnum—oui

Pour la colonne role, ajouter les valeurs autorisees : administrateur et gestionnaire.

4. Fichier de configuration local

Renseigner src/main/resources/config.properties avec les informations recuperees :

# Configuration Appwrite Cloud
APPWRITE_ENDPOINT=https://fra.cloud.appwrite.io/v1
APPWRITE_PROJECT_ID=ton_project_id
APPWRITE_API_KEY=ta_cle_api
APPWRITE_DATABASE_ID=ton_database_id

TABLE_PRODUIT=produit
TABLE_FOURNISSEUR=fournisseur
TABLE_COMMANDE=commande
TABLE_UTILISATEUR=utilisateur
Attention : ne pas committer la cle API dans Git. Le fichier est ignore via .gitignore.

5. Creer le premier administrateur

Generer un hash BCrypt pour le mot de passe choisi :

mvn -q compile exec:java \
  -Dexec.mainClass=com.magasin.outils.GenererHashAdmin \
  -Dexec.args="motDePasseAdmin"

La commande affiche une chaine du genre $2a$12$xxxx.... Copier ce hash, puis dans la console Appwrite :

  1. Ouvrir la table utilisateur.
  2. Cliquer sur Create row.
  3. Remplir : username = admin, password = $2a$12$xxxx..., role = administrateur.

6. Compiler et lancer

mvn clean package
java -jar target/gestionnaire-stock-magasin.jar

Ou en mode developpement, sans empaqueter :

mvn compile exec:java

7. Reglages HiDPI (ecran haute densite)

Par defaut l'app force une mise a l'echelle 2x sur Linux. Pour ajuster :

MAGASIN_UI_SCALE=1.5 java -jar target/gestionnaire-stock-magasin.jar
# ou
java -Dsun.java2d.uiScale=1.0 -jar target/gestionnaire-stock-magasin.jar
